# enSmart conversation agent a Home Assistant Assist-hoz

Ha nincs beállítva system prompt akkor az API oldalon konfigurált system prompt-ot fogja használni. Ha van beállítva system prompt, akkor azt hozzáadja az API oldalon beállított system prompt-hoz.

Alapértékek (ha az integrációba nincsenek beállítva akkor API oldalon ezek az értékek érvényesülnek):
`Temperature = 0.2` 
`Top P = 1`
